# 🤖 EduNova AI - Student Assistant Chatbot

EduNova AI is a beginner-friendly AI chatbot web app built using **Python** and **Streamlit**.  
It can understand simple student-related queries, reply with intelligent responses, and even remember user information during conversation.

This project helped me learn:
- NLP basics
- Chatbot architecture
- Intent recognition
- Streamlit deployment
- Conversational AI concepts

---

# 🚀 Features

✅ Interactive AI chatbot  
✅ Intent-based response system  
✅ Randomized responses  
✅ Conversational memory  
✅ Streamlit web interface  
✅ Beginner-friendly architecture  

---

# 🧠 Technologies Used

- Python
- Streamlit
- Random Library
- Session State Memory

#⚡ Installation

Clone this repository:

git clone https://github.com/yourusername/EduNova-AI.git

Go into project folder:

cd EduNova-AI

Install dependencies:

pip install streamlit

Run the application:

streamlit run app.py
----

Example Interaction
You: hi
Bot: Hello 👋

You: what is ml
Bot: Machine Learning is AI that learns from data.

You: my name is mithra
Bot: Nice to meet you, mithra!

You: what is my name
Bot: Your name is mithra
